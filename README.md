# Бот-пиццерия

> Бот-пиццерия - здесь вы сможете выбрать пиццу и напитки, добавить их в корзину и оформить заказ.


### Запуск проекта

Находясь в корневой директории проекта выполните команду в терминале

```
python -m app.main
```
